부등호, 동등, 더하기

부등호

  1. 참조형 타입을 기본형으로 변경
  2. 두 피연산자가 문자열이라면 문자열 비교 'a' < 'b'
  3. 그렇지 않다면 숫자로 변경 후 비교

x <= y 이면 x < y 이거나 x = y이어야 한다. (수학)

x = 0 , y = null 이라면 x <= y ( 0 <= 0 ) 이지만 x < y 와 x = y가 거짓일 수 있다 (자바스크립트)

더하기 연산자
참조형은 기본형으로
피 연산자 둘중 하나가 문자열이라면
문자 더하기를 진행
그렇지 않다면 두 값을 문자로 변경하여 더하기를 진행

동등 == 이 참이 경우 : 일치하면~~, undefined, null, 참조형은 같은 객체를 참조해야한다. 타입이 같다면 값이 같아야한다. 타입이 다르다 숫자로 변경하고 일치 비교한다.
